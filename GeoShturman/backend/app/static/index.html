Как это работает

  1. Dead Reckoning
  2. Рельеф из baro − radar
  3. Particle Filter
  4. Profile Matching с DEM
  5. Координаты · курс · confidence

Алгоритм сначала ведёт БПЛА по Dead Reckoning: скорость, курс и время дают прогноз движения без GNSS. Затем радиовысотомер и барометр восстанавливают профиль рельефа по формуле terrain = baro − radar. Particle Filter хранит облако гипотез положения, сравнивает каждую с DEM по текущей высоте и окну профиля, пересчитывает веса частиц и выдаёт координаты, скорость, курс, радиус ошибки и confidence.
